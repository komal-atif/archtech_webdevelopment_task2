# ✨ Pink To-Do List Application

A beautiful, modern to-do list application with a pink theme, perfect for organizing tasks in style! This application features a split-screen design with interactive elements, animations, and image sharing capabilities.

## 🎯 Features

### 🎨 **Beautiful Design**

- Pink-themed gradient background with floating animations
- 7 different color themes for your lists
- Smooth animations and transitions
- Responsive design for all devices

### 📝 **Task Management**

- Create custom to-do lists with names
- Add, edit, and delete tasks
- Mark tasks as complete with checkboxes
- Temporary task list before creating final list

### 💾 **Data Persistence**

- Save lists to browser's localStorage
- Load previously saved lists
- Delete saved lists when no longer needed
- All data persists between browser sessions

### 📱 **Sharing Features**

- Download lists as beautiful images
- Optimized for WhatsApp sharing
- Maintains UI design in shared images
- Includes all tasks in the image

### 🎯 **User Experience**

- Split-screen layout (input on left, display on right)
- Scrollable task lists
- Always visible action buttons
- Intuitive interface with visual feedback
